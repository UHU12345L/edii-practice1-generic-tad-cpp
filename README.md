# ED II - Practice 1: Generic TAD in C++

Course: Estructuras de Datos II
University: Universidad de Huelva
Year: 2025-2026

## 📄 Description
Implementation of a generic Multiset TAD (TAD Multiconjunto) in C++
using templates. Includes the full implementation and a user program
to test the data structure.

## 📁 Structure
- `multiconjunto/` — TAD implementation (header + source)
- `usuario/` — Main program using the TAD

## 💻 Technologies
- Language: C++
- IDE: Code::Blocks
